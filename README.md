# ForestGuard IoT Monitoring System

A smart forest monitoring project using IoT, sensors, and real-time communication for environmental risk detection.

## Overview
This project explores how connected devices and sensor-based systems can support early detection and monitoring in remote natural environments.

## Main ideas
- Environmental monitoring
- IoT-based sensing
- Real-time data transmission
- Scalable monitoring systems

## Notes
This repository is based on an existing ForestGuard project and reorganized/presented here as part of my technical portfolio.
